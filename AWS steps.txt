1. npm install -g @aws-amplify/cli

2. amplify configure

3. configure will open console, navigate to IAM user creation and create a new user

4. Set permissions for the new user "AdministratorAccess-Amplify"

5. Click on the new user and on the  Security credentials tab, select Create access keys.

6. Selecte Command line interface for access keys

7. Download the csv files which has secret keys and access keys.

8. Enter the access keys and secret keys values you just copied into the corresponding CLI prompts.


##### After Authentication front end ready ####

9. npm install aws-amplify

10. amplify init

11. amplify add auth

12. amplify push -y

#### For Storage ####

13. amplify add storage

14. amplify push -y
